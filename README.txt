README - MBG Attendance (client-side demo)
- Cara pakai: ekstrak ZIP -> buka index.html pake browser (Chrome/Edge/Firefox).
- Login demo: admin / admin123
- Ketua kelas: username 'ketua_X-1' ... 'ketua_XII-10' dengan password 'ketua_1pass'... contoh pada file app.js

- Jangan simpan password dalam teks biasa di produksi.
